# webpacksvgsprite

### This a POC 

This is an example of an icon es6 Javascript module using webpack `svgo-loader` to clean svg files and `svg-sprite-loader` to create svg an sprite.

https://github.com/kisenka/svg-sprite-loader

https://github.com/rpominov/svgo-loader

## Install

`npm install`

`npm run dev`
